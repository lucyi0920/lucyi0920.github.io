Project: Global Content Delivery Platform

Project Documentation

View Full PDF

Project Overview

Developed an advanced hybrid cloud architecture for Company A, addressing scalability, latency, and global delivery challenges. This proof-of-concept project demonstrates a high availability, secure, full-stack infrastructure for seamless media delivery.

Problem Statement

  • High latency in content delivery, affecting user experience
  • Scalability limits with on-premises infrastructure
  • Complex multi-region connectivity needs
  • High availability requirements for apps and databases

Solution Overview

  • Network Architecture: Transit Gateway peering for cross-region communication between Seoul and São Paulo
  • High Availability: Multi-AZ VPCs with RDS read replicas for resilient data handling
  • Hybrid Connectivity: Site-to-Site VPN for secure on-premises integration
  • Optimized Content Delivery: Global Accelerator and Route 53 for latency reduction and global availability
  • Serverless & Multi-language: Built with Amplify, Lambda, and API Gateway for scalable content processing
  • Automated Deployment: CloudFormation templates for consistent infrastructure setup
  • Security: AWS WAF, IAM, and ACM for multi-layered protection
  • Monitoring: CloudWatch and SNS for proactive system health tracking

Technologies Used

AWS VPC, Transit Gateway, RDS, Global Accelerator, Route 53, Lambda, API Gateway, Amplify, CloudFormation, WAF, CloudWatch, SNS

First AWS project developed as part of the K-Digital Training program at Soldesk (AWS and Azure Hybrid/Multi-Cloud DevOps Course)

Project: AI-Powered Short-Form Video Generator

Project Documentation

View Full PDF

Project Overview

Developed an AI-powered short-form video creation platform utilizing AWS cloud services, addressing the challenges of time-consuming content creation and the need for tailored marketing materials. This solution enables quick, automated video generation while maintaining high quality and personalization.

Problem Statement

  • High costs and time investment in short-form content creation
  • Difficulty in producing personalized content for target audiences
  • Limited multi-language support for global market reach
  • Complex content creation process for small businesses

Solution Overview

  • AI Integration: Leveraging Amazon Bedrock and Rekognition for automated content generation and analysis
  • Containerized Architecture: Using ECS and Docker for scalable application deployment
  • Serverless Processing: Lambda functions for efficient video processing and content handling
  • Infrastructure as Code: Terraform for consistent and reproducible deployments
  • Security: IAM and CloudWatch for comprehensive security monitoring
  • Data Management: DynamoDB for user data and content metadata storage

Key Achievements

  • 90% reduction in content creation time
  • Cost optimization through serverless architecture
  • Enhanced content personalization capabilities
  • Improved accessibility for small businesses

Technologies Used

AWS Bedrock, Rekognition, ECS, ECR, Lambda, API Gateway, DynamoDB, CloudWatch, SNS, Docker, Terraform

Final project developed as part of the K-Digital Training program at Soldesk (AWS and Azure Hybrid/Multi-Cloud DevOps Course)
